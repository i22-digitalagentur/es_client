# ESClient

[![Build Status](https://travis-ci.org/tlux/es_client.svg?branch=master)](https://travis-ci.org/tlux/es_client)

A minimalistic Elasticsearch client for Elixir.

## Prerequisites

* Elixir >= 1.8
* Erlang >= 20
